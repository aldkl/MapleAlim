시작 가이드

캐릭터 추가 전 확인
메이플 플랜은 등록한 캐릭터를 기준으로 주간 보스 계획, 실제 처치 체크, 일퀘/주간퀘 완료율을 관리합니다. 처음에는 캐릭터를 하나 추가하고, 그 다음 보스와 TODO를 캐릭터별로 맞추면 됩니다.
STEP 1
로컬 서버로 접속

캐릭터 검색은 브라우저 파일 열기가 아니라 로컬 서버 주소에서 동작합니다.

  • PowerShell에서 `python .\server.py` 실행
  • 브라우저에서 `http://127.0.0.1:8765/` 접속
  • API 키는 `.env`에 보관하고 화면에는 노출하지 않음
http://127.0.0.1:8765/
STEP 2
캐릭터 추가

상단 검색창에 캐릭터명을 입력하면 Nexon API에서 기본 정보를 가져옵니다.

  • 캐릭터명, 월드, 직업, 레벨, 이미지 표시
  • 같은 계정/월드 주간 보스 90개 제한 계산에 활용
  • 캐릭터 이름을 드래그해서 표시 순서 변경
캐릭터 정산
STEP 3
보스와 TODO 맞추기

잡을 보스를 먼저 고르면 TODO 화면에서 실제 처치 여부를 체크할 수 있습니다.

  • 주간 보스에서 캐릭터별 보스와 파티 인원 설정
  • TODO에서 실제 처치, 일퀘, 주간퀘 체크
  • 안 하는 캐릭터는 일퀘/주간퀘/보스 제외 가능
/weekly-boss · /todo

오늘/이번 주 숙제
계정 전용 숙제는 같은 메이플 계정에서 공유됩니다. 에픽던전은 각 캐릭터의 주간퀘에 표시되며 캐릭터별로 하나만 선택할 수 있습니다.
1. 사냥 캐릭터와 장비
Nexon Open API 장비 프리셋 1~3의 잠재옵션을 비교해 드롭률+메획 합계가 가장 큰 프리셋을 자동 적용합니다. 직접 수정도 가능합니다.
추가 획득 설정
재획비, 유니온, 아티팩트, 홀리 심볼 등 장비 외 효과의 합계를 입력하세요.
2. 사냥 기준
몬스터 1마리 기본 메소는 `몬스터 레벨 × 7.5`, 메소 주머니 기본 드롭률은 60%로 가정하며 아이템 드롭률의 영향을 받습니다. 메소 제한은 직접 조정할 수 있습니다.
드롭 기대값 기준
공식 드롭 확률은 공개되어 있지 않아 기본 확률을 코어 젬스톤 1/4,000, 솔 에르다 조각 1/5,000로 둔 추정값입니다. 실제 기록에 맞춰 수정할 수 있으며 드롭률 배율을 적용합니다.
3. 메소 제한까지 예상 수익
캐릭터와 사냥 기준을 입력하면 자동 계산됩니다.
